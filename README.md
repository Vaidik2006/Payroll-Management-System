A web-based Payroll Management System built using **Python (Flask)** that automates employee salary computation, payslip generation, and payroll visualization.

## Features
- Secure Admin and Employee Login
- Add, Edit, Delete Employee Records
- Automatic Salary Computation (HRA, DA, PF, Tax, Loan)
- Payslip Generation & PDF Download
- Dynamic Charts for Payroll Trends
- JSON-based Data Storage

## Tech Stack
- **Frontend:** HTML, CSS (Dark Theme)
- **Backend:** Flask (Python)
- **Data Storage:** JSON
- **Visualization:** Chart.js

pip install flask
